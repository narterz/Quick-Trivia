This small quiz app tests the users ability to answer questions from a variety of categories. Questions are randomly generated from the Trivia APi at https://the-trivia-api.com/.

This projects demonstates the follwing frontend skills
  - State and props management
  - REST API's
  - Axios
  - API data management
  - React-Router-V6
  - React bootstrap
  - useRefs
  - Scss mixins
  - Scss functions
  - Scss maps
  - Scss file management
  - Responsive design for mobile, tablet, desktop
  - UI / UX design
